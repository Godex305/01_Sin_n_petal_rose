# 🌹 Sine Rose Curve Animation using Manim

This repository contains the source code for a mathematical animation demonstrating the **Sine Rose Curve** using **parametric equations** in Python with Manim.

---

## 📌 About the Project

This project visualizes the beautiful and symmetric **rose curve** generated by the sine function in polar/parametric form.

The animation is designed to help students understand how changing parameters affects the shape and number of petals in the curve.

---

## 📐 Mathematical Concept

The sine rose curve is defined as:

\[
r = a \sin(n\theta)
\]

- `a` → controls the size of the curve  
- `n` → determines the number of petals  
  - If `n` is odd → number of petals = `n`  
  - If `n` is even → number of petals = `2n`  

---

## 🎥 What This Animation Shows

- Step-by-step plotting of the curve  
- Smooth parametric tracing of the graph  
- Visualization of different values of `n`  
- Symmetry and periodic nature of the curve
